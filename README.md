# LiRu
## Line Follower In Rust

A line-following robot controlled by an STM32F401RE microcontroller running Rust firmware, with wireless control and telemetry via Bluetooth.

## Hardware Components

- Nucleo F401RE (STM32F401RE microcontroller)
- DRV8833 (Dual H-bridge motor driver)
- HY-S301 (8-Channel Line Tracking Sensor Module)
- HC-05 (Bluetooth module)
- Pololu micro DC motors (x2)
